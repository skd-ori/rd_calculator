# RD 計算機 (Web 版) 動作確認 V4

## 概要

`docs/index.html` に単一ファイル構成の RD 計算機を実装しました。
(旧パス: `docs/rd_calculator/index.html` から移動しました)

**V4 アップデート内容:**

- **技威力データの修正**: 威力 85-V3(真) の値を 374 → **375** に修正しました。
- **RD 計算ロジックの変更**: DF(防御側)が AT(攻撃側)を上回った場合、RD 値が**マイナス**で表示されるようになりました。
- **パッシブ補正**: 入力が空欄または 0 の場合は、補正なし(100%効率)として計算されます。

## 使用方法

1. GitHub Pages URL または `docs/index.html` を開く。
2. モードを選択する。
3. **技威力 (AT/DF)**:
   - 「ベース威力」と「進化段階」をボタンで選択。
   - 例: ベース 85 + 進化 V3(真) = 375
4. ステータス値、パッシブ補正を入力。
5. 「計算する」ボタンを押して結果を確認。
   - **負の RD 値**が表示されることを確認してください (DF > AT の場合)。

## 勝率判定ロジック

- **フォーカスモード**: RD 値のみ表示。
- **その他**:
  - **RD 15%以上**: 100% 勝利/敗北
  - **RD 14.6%**: 55% 勝率
  - **RD 0%**: 40% 勝率
