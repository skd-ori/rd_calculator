# RD 計算機 (Web 版) 動作確認 V3

## 概要

`docs/rd_calculator/index.html` に単一ファイル構成の RD 計算機を実装しました。
V3 更新:

- **技威力 (Technique Power)** の入力方法をボタン選択式に変更しました。
- 「ベース威力」と「進化段階」を選択することで、Excel 数式に基づく正しい技威力が自動入力されます。

## 使用方法

1. `docs/rd_calculator/index.html` を Web ブラウザで開く。
2. モードを選択する。
3. **技威力 (AT/DF)**:
   - 「ベース威力」ボタン (30, 50, 60, 70, 85, 100) をクリック。
   - 「進化」ボタン (無〜GO) をクリック。
   - → 自動的に技威力が計算され、グレーのボックスに表示されます。
4. ステータス値、パッシブ補正を入力。
5. 「計算する」ボタンを押して結果を確認。

## 勝率判定ロジック

- **フォーカスモード**: RD 値のみ表示。
- **その他**:
  - **RD 15%以上**: 100% 勝利/敗北
  - **RD 14.6%**: 55% 勝率
  - **RD 0%**: 40% 勝率
